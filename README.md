# Cyber Snake Minigame

Cyber Snake is a small cyberpunk-inspired Snake minigame built with HTML, CSS and JavaScript.

The project was created as a creative web experiment, combining classic arcade gameplay with neon visuals, particles, responsive controls and native browser audio.

## Live Demo

Play here:  
https://eduardovoigtdev.github.io/cybersnake-minigame/

## Features

- Classic Snake gameplay
- Cyberpunk / neon visual style
- HTML Canvas rendering
- Score and high score system
- High score saved with localStorage
- Keyboard controls
- Mobile touch controls
- Particle effects
- Native JavaScript audio using Web Audio API
- Responsive layout

## Controls

### Desktop

- `W` / `Arrow Up` — move up
- `A` / `Arrow Left` — move left
- `S` / `Arrow Down` — move down
- `D` / `Arrow Right` — move right
- `Space` — start, pause or restart
- `Esc` — pause

### Mobile

Use the on-screen directional buttons.

## Technologies

- HTML5
- CSS3
- JavaScript
- Canvas API
- Web Audio API
